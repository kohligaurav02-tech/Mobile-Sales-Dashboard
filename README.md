# 📱 Mobile Sales Dashboard | Power BI

## 📌 Project Overview

The Mobile Sales Dashboard is an interactive Power BI project designed to analyze mobile sales performance across multiple business dimensions. It provides business insights through interactive visualizations, KPIs, maps, and dynamic filters.

### 🎯 Objectives
- Monitor overall sales performance.
- Analyze sales by city, brand, and payment method.
- Track Month-to-Date (MTD) performance.
- Compare current sales with the same period last year (YoY).
- Enable interactive analysis using slicers and filters.
